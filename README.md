Vào folder fe, ở hộp chứa tên tập tin nhập cmd để mở Command Prompt lên. 
Copy dòng dưới này vào trong Command Prompt để cài đặt các package mà không tuân thủ nghiêm ngặt các ràng buộc về peer dependencies được định nghĩa trong các package đó. 
   npm install --legacy-peer-deps 
Sau khi install xong thì nhập lệnh npm run start để chạy fe.
